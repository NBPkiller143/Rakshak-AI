# 🛡️ Rakshak AI

> AI-Powered Intelligent School Security & Surveillance System

---

## 📌 Overview

Rakshak AI is an AI-powered intelligent surveillance system designed to enhance school security through real-time monitoring, object detection, automated incident reporting, and emergency response. The system combines Computer Vision, Artificial Intelligence, Robotics, and a modern web dashboard into a single security platform.

---

## ✨ Features

- 🔐 Secure Login & Logout
- 🎥 Live CCTV Monitoring
- 🧠 YOLOv8 Object Detection
- 👥 Real-Time Person Counting
- 🤖 AI Robot Dispatch
- 🚨 Emergency Mode
- 🎤 Voice Assistant
- 📸 Snapshot Capture
- 🎥 Video Recording
- 📄 AI Incident Report Generation (PDF)
- 🗄️ SQLite Database Logging
- 📊 Real-Time Dashboard
- 🔔 Smart Notifications
- 📍 Campus Map Monitoring
- ⚙️ Settings with Local Storage
- 🌙 Dark Mode
- 🎮 Demo Mode

## 🚀 Installation

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python app.py
```

Open your browser

```text
http://127.0.0.1:5000
```
